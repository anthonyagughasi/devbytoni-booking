# DevByToni Booking Pro

A lightweight, secure WordPress booking plugin with time slots, PayPal payments, admin calendar view, and email notifications.

## Features
- Frontend booking form with datepicker and time slots
- PayPal integration for paid bookings
- Automatic email confirmations
- Admin calendar and booking management
- Optional Google reCAPTCHA
- Prevents double bookings

## Installation
1. Upload the `devbytoni-booking` folder to `/wp-content/plugins/`
2. Activate in WordPress admin
3. Configure settings under Settings > Booking Settings
4. Add `[devbytoni_booking_form]` shortcode to a page
